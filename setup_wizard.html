۱ تلگرام (اجباری)
TELEGRAM_BOT_TOKEN

TELEGRAM_CHAT_ID

۲ RapidAPI — JSearch (اختیاری)
💡 بدون این کلید، فقط منابع رایگان استفاده میشه.
از [RapidAPI](https://rapidapi.com) ثبت‌نام کن و JSearch API رو جستجو کن.
RAPIDAPI_KEY
کلید وارد نشده
۳ Cloudflare Worker (اختیاری)
💡 اگه راه‌اندازی کنی، آگهی‌های Remote OK و We Work Remotely هم اضافه میشه.
آدرس Worker رو از [dash.cloudflare.com](https://dash.cloudflare.com) کپی کن.
CF_WORKER_URL
آدرس وارد نشده
۴ پروفایل کاربر (اختیاری)
💡 مهارت‌هات رو وارد کن تا ربات آگهی‌های مرتبط‌تر رو پیدا کنه.
USER_SKILLS
مهارت‌ها وارد نشده (پیش‌فرض استفاده میشه)
۵ Cover Letter با ChatGPT (اختیاری)
🤖 هر آگهی یک دکمه "ChatGPT Cover Letter" داره.
کلیک → باز شدن ChatGPT با پرامپت آماده.
⚠️ نیازی به API key نیست — رایگان از اکانت ChatGPT خودت استفاده می‌کنی.
CL_PROMPT
پرامپت وارد نشده (پیش‌فرض استفاده میشه)
